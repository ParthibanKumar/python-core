# Python Core

Foundational Python practice, part of a structured transition from
Ab Initio ETL development to modern data engineering.

## Setup

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

## Structure

| Folder | Purpose |
|---|---|
| `src/` | Scripts, one module per topic |
| `tests/` | Unit tests |
| `data/` | Input files (not tracked) |
| `notes/` | Daily learning log |
